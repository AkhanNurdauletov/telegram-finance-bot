Telegram Finance Bot

Telegram-бот на Python с использованием `aiogram`, предназначенный для личного учёта доходов и расходов.

Функции

- Добавление доходов и расходов через Telegram
- Просмотр истории операций
- Удаление и редактирование записей
- Экспорт данных в Excel
- SQLite база данных
- FSM (машина состояний) для пошагового добавления записей

Стек

- Python 3.10+
- aiogram 3.2
- SQLite
- pandas + openpyxl
